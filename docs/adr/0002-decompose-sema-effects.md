# ADR 0002: Decompose content, addressing, and enforcement

- Status: accepted
- Date: 2026-07-14

## Decision

Use a five-condition ladder with an equal-information prose arm and an opaque-ID
resolver control before calling an observed difference a Sema effect.

## Rationale

A task-only baseline versus a rich Pattern Card conflates better instructions
with content-addressed identity. A Sema reference without a compliant runtime
also cannot establish the benefit of fail-closed enforcement.

## Consequences

Experiments cost more runs, but each important contrast maps to a specific
mechanism. Reports must identify which contrast supports each claim.
